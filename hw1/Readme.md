# ДЗ 1: Классификация изображений на CIFAR-10

## Описание задачи

**Задача:** Множественная классификация изображений

Необходимо обучить сверточную нейронную сеть для классификации изображений на 10 классов. В рамках данного задания использовалась архитектура **EfficientNet-B2**.

## Датасет: CIFAR-10

CIFAR-10 (Canadian Institute For Advanced Research) — известный датасет для классификации изображений, часто используемый для тестирования алгоритмов машинного обучения.

**Характеристики датасета:**

* Количество изображений: 60,000 цветных изображений

  * Train: 50,000 изображений
  * Test: 10,000 изображений
* Размер изображений: 32×32 пикселя
* Количество каналов: 3 (RGB)
* Количество классов: 10

**Классы:**

* airplane
* automobile
* bird
* cat
* deer
* dog
* frog
* horse
* ship
* truck

**Распределение классов:**
Датасет сбалансирован — каждый класс содержит ровно 5,000 обучающих изображений и 1,000 тестовых изображений.

**Источник:**

* [Официальный сайт CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html)
* Также датасет доступен через `torchvision.datasets.CIFAR10`

## Архитектура: EfficientNet-B2

**EfficientNet-B2** — современная сверточная нейронная сеть, использующая **compound scaling** для оптимального баланса между точностью и эффективностью вычислений.

После адаптации для CIFAR-10 EfficientNet-B2 имеет **~7.7M параметров**, что делает её более компактной по сравнению с более старыми архитектурами.

**Адаптация для CIFAR-10:**

* Первый слой: `stride=1` вместо `stride=2`
* Убран max pooling после первого слоя
* Классификатор адаптирован для 10 классов

**Преимущества:**

* Современная архитектура с высокой эффективностью
* Компактная, меньше параметров, чем у традиционных сетей
* Хорошо подходит для экспериментов и модификаций
